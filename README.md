## 📚 Contenido: 5 Talleres Estructurados

### **TALLER 01 - Tipos de Datos Básicos en Python** | 1-2 horas

**Conceptos:**
Tipos primitivos (`int`, `float`, `str`, `bool`), estructuras fundamentales (`list`, `tuple`, `dict`)

**Objetivos:**
- Identificar y manipular tipos de datos en Python
- Elegir estructuras de datos apropiadas según contexto
- Aplicar nomenclatura profesional (snake_case, camelCase)

**Métodos:**
- Ejercicios secuenciales
- Análisis comparativo de estructuras
- Buenas prácticas en codificación

---

### **TALLER 02 - Entrada/Salida y Manipulación de Archivos** | 2 horas

**Conceptos:**
Captura de entrada (`input()`), conversión de tipos, operaciones de archivo (`read`, `write`, `append`)

**Objetivos:**
- Crear aplicaciones interactivas con el usuario
- Manipular archivos de texto (.txt)
- Gestionar errores en I/O
- Implementar persistencia de datos

**Métodos:**
- Contexto `with open(...)`
- Manejo de excepciones
- Ejercicios con datasets reales

---

### **TALLER 03 - DataFrames y Procesamiento de Datos con Pandas** | 2-3 horas

**Conceptos:**
Tipado dinámico Python, librerías científicas (Pandas, NumPy), carga/exploración de CSV

**Objetivos:**
- Cargar y explorar datasets desde archivos
- Identificar y tratar valores faltantes
- Aplicar técnicas de imputación (media, mediana, moda)
- Preparar datos crudos para análisis

**Métodos:**
- Métodos `iloc`, `loc`, slicing
- Técnicas de limpieza de datos
- Visualización básica

**Tecnologías:** `pandas`, `numpy`, `matplotlib`, `Google Colab`

---

### **TALLER 04 - Tipos de Datos Avanzados: Sets (Conjuntos)** | 1-2 horas

**Conceptos:**
Conjuntos en Python, operaciones de lógica de conjuntos (intersección, diferencia, unión)

**Objetivos:**
- Crear y manipular sets
- Realizar operaciones de teoría de conjuntos
- Aplicar en problemas prácticos (gestión de usuarios, listas únicas)
- Optimizar búsquedas de elementos únicos

**Métodos:**
- Métodos nativos: `.intersection()`, `.union()`, `.difference()`
- Operadores simbólicos: `&`, `|`, `-`
- Casos de uso real

---

### **TALLER 05 - Estructuras de Control: Condicionales e Iteración** | 2-3 horas

**Conceptos:**
Lógica condicional (`if/elif/else`), bucles (`for`, `while`), funciones, algoritmos secuenciales

**Objetivos:**
- Escribir algoritmos que combinen control secuencial, condicional e iterativo
- Procesar colecciones de datos mediante iteración
- Encapsular lógica en funciones reutilizables
- Aplicar en problemas de clasificación y automatización

**Métodos:**
- Control de flujo avanzado
- Comprensión de listas (list comprehensions)
- Diseño de funciones con múltiples retornos
- Algoritmos prácticos
