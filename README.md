# 📚 Base de Datos

Repositorio de proyectos y ejercicios relacionados con **Bases de Datos**. Aquí encontrarás scripts SQL, modelos de datos, procedimientos almacenados, y proyectos prácticos.

---

## 📁 Estructura del Repositorio

```
📦 software_y_base_de_datos
 ┣ 📂 SQL
 ┃ ┣ 📂 DDL              → Scripts de Definición de Datos (CREATE, ALTER, DROP)
 ┃ ┣ 📂 DML              → Scripts de Manipulación de Datos (INSERT, UPDATE, DELETE)
 ┃ ┣ 📂 Consultas        → Consultas SELECT (simples, joins, subconsultas)
 ┃ ┣ 📂 Vistas           → Definición de Vistas (VIEW)
 ┃ ┣ 📂 Procedimientos_Almacenados → Stored Procedures
 ┃ ┣ 📂 Funciones        → Funciones definidas por el usuario
 ┃ ┣ 📂 Triggers         → Disparadores
 ┃ ┗ 📂 Indices          → Índices y optimización
 ┣ 📂 NoSQL
 ┃ ┣ 📂 MongoDB          → Scripts y ejemplos con MongoDB
 ┃ ┗ 📂 Redis            → Scripts y ejemplos con Redis
 ┣ 📂 Modelado
 ┃ ┣ 📂 Diagramas_ER     → Diagramas Entidad-Relación
 ┃ ┗ 📂 Normalizacion    → Ejercicios de Normalización (1FN, 2FN, 3FN, BCNF)
 ┗ 📂 Proyectos          → Proyectos integradores de Base de Datos
```

---

## 🛠️ Tecnologías

- **SQL**: MySQL / PostgreSQL / SQL Server
- **NoSQL**: MongoDB, Redis
- **Herramientas de modelado**: MySQL Workbench, draw.io

---

## 🚀 Cómo usar este repositorio

1. Navega a la carpeta del tema que te interese.
2. Abre el archivo `.sql` o `.md` correspondiente.
3. Ejecuta los scripts en tu gestor de base de datos favorito.

---

## 📖 Contenido

### SQL
| Carpeta | Descripción |
|--------|-------------|
| `DDL` | Creación y modificación de tablas, esquemas y restricciones |
| `DML` | Inserción, actualización y eliminación de datos |
| `Consultas` | Consultas simples, con JOIN, subconsultas y agrupaciones |
| `Vistas` | Creación y uso de vistas |
| `Procedimientos_Almacenados` | Stored Procedures con lógica de negocio |
| `Funciones` | Funciones escalares y de tabla |
| `Triggers` | Disparadores para automatización |
| `Indices` | Creación de índices para optimización de consultas |

### NoSQL
| Carpeta | Descripción |
|--------|-------------|
| `MongoDB` | Operaciones CRUD y agregaciones en MongoDB |
| `Redis` | Estructuras de datos y comandos Redis |

### Modelado
| Carpeta | Descripción |
|--------|-------------|
| `Diagramas_ER` | Diagramas Entidad-Relación de distintos dominios |
| `Normalizacion` | Ejercicios prácticos de normalización de bases de datos |
